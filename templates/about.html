Our Project

The Technology Behind the Analysis

A deep learning nutrition analyzer built with MobileNetV2, Flask, and WHO/ICMR dietary standards.

How It Works

Architecture & Methodology

01

Technical Architecture

Backend Framework: Flask (Python Web Framework)

Frontend: HTML5, CSS3, JavaScript with Chart.js

Deep Learning: TensorFlow/Keras with MobileNetV2

Data Storage: CSV-based Nutritional Database

02

Deep Learning Model

Architecture: MobileNetV2 - Lightweight and efficient

Training Method: Transfer Learning on pre-trained ImageNet weights

Food Categories: 7 trained classes (Amul Milk, Curd, Kit Kat, Lays Chips, Maggi, Oreo, Parle G)

Input Size: 224×224 pixels, RGB format

03

Image Classification Process

Step 1: User uploads food product image

Step 2: Image preprocessed: resized to 224×224, normalized to [0,1]

Step 3: MobileNetV2 model predicts food category with confidence

Step 4: Results returned in real-time with confidence percentage

04

Nutrition Data Retrieval

Data Source: CSV database with 7 food items

Schema: food_name, net_weight_g, calories_per_100, protein_g_per_100, fat_g_per_100, carbs_g_per_100, sugar_g_per_100, sodium_g_per_100

Quantity Engine: Per Serving uses net_weight_g with serving stepper, and Custom mode supports any gram input

Real-Time Update: All displayed nutrition values are recalculated instantly for the selected quantity

05

Rule-Based Health Verdict (ICMR/WHO)

Nutrient Rules: Protein, fat, carbs, sugar, and sodium are each classified as Healthy/Moderate/Unhealthy

Final Verdict: Total nutrient score (0-10) maps to Healthy (>=8), Moderate (5-7), or Unhealthy (<=4)

Dairy Exception: For Amul Milk and Curd, fat thresholds are relaxed and natural lactose is excluded from sugar penalty

Transparency: A per-nutrient reason table explains every verdict on the dashboard

06

Formula-Based Health Score

Score Formula: Uses macro calorie percentages, fat and sugar penalties, sodium penalty, and carb balance

Range: Clamped to 0-100 with color bands for Healthy, Moderate, and Unhealthy

Explainability: Component breakdown shows contribution of Protein, Fat, Sugar, Sodium, and Carbs

Dairy Handling: Dairy items apply fat penalty reduction and no sugar penalty for natural lactose
